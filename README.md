# Hired---job-tracker-app
Hired — Job Search Optimizer (Pro)
===================================

Getting Started
---------------
1. Extract the ZIP file to any folder on your computer.
2. Open index.html in your browser (Chrome, Firefox, Safari, or Edge).
3. Start using the app — no internet connection required.

What's Included in Pro
-----------------------
✓ Unlimited job tracking
✓ Strategy Coach — weekly personalised recommendations
✓ Failure Analysis — understand exactly why you're getting rejected
✓ Advanced Analytics — conversion funnel, source performance, weekly targets
✓ Resume Performance Tracking — compare response rates per CV version
✓ Job Match Scoring — auto-scores jobs against your skills
✓ Company Intelligence — track patterns and response speed per company
✓ Export Insight Report — full PDF/print-ready job search report
✓ Dark & Light mode
✓ Works 100% offline — your data stays on your device

Tips
----
• Click "⚡ Demo" in the top bar to load sample data and explore all features.
• Your data is saved automatically in your browser's localStorage.
• Use "↓ Export Report" to save a snapshot of your job search progress.
• Use "↺ Reset Data" (bottom of sidebar) to start fresh at any time.

Support
-------
All features are client-side and require no login or backend.
Data persists in localStorage — clearing browser data will reset the app.
